# Item Manage TODO / Roadmap

今後追加・改善を検討している機能の一覧です。

## Priority 1: 安定化と基本機能の完成
- [ ] **DB 整合性の確保**: マイグレーション管理。
- [x] **バリデーションの強化**: 入力値（価格が数値であるか、ISBNが有効かなど）のチェック。
- [x] **エラーハンドリング**: スキャン失敗時やAPIエラー時の通知。

## Priority 2: ユーザー体験（UX）の向上
- [x] **検索・フィルタリング**: キーワード、カテゴリ、タグによる検索。
- [ ] **並び替え機能**: 登録日順、名称順、価格順でのソート。
- [ ] **モバイル体験の最適化**: バーコードリーダーの読み取り精度向上。
- [ ] **デザインのブラッシュアップ**: ガラスモーフィズムの調整。

## Priority 3: 機能拡張
- [x] **カテゴリ管理**: 本以外のカテゴリ（CD、DVD、ゲーム等）に対応し、楽天APIを用いた自動取得・推定機能を実装。
- [ ] **統計ダッシュボード**: 所有アイテムの総額表示、カテゴリ別グラフ。
- [ ] **エクスポート機能**: CSV や JSON 形式でのデータ書き出し。
- [ ] **一括編集・削除**: 複数のアイテムを選択して一括で処理。

## Priority 4: インフラ・高度な機能
- [ ] **複数画像対応**: アイテムに複数の写真をアップロード・保存可能にする（R2連携）。
- [x] **外部API連携の拡大**: 楽天ブックス総合検索APIとの連携を実装。
- [ ] **PWA化**: オフラインでもリストの確認ができるようにする。
